# 第10章 金融风险分析与模拟（章节文档）

## 1. 章节定位

本章从“收益导向”转向“收益-风险联合评估”，建立风险量化与情景思维。

## 2. 目录结构

1. 波动率与风险度量
2. Monte Carlo 模拟基础
3. 情景分析与压力测试
4. 案例：资产收益模拟分析
5. 本章小结与风险沟通

## 3. 内容介绍

### 3.1 波动率与风险度量
- 讲解历史波动率、下行风险等基础度量。
- 引入风险指标在投资决策中的解释方式。
- 讨论短样本与结构变化对风险估计的影响。

### 3.2 Monte Carlo 模拟
- 说明随机抽样、路径生成、分布假设等核心步骤。
- 实现资产价格或收益路径模拟。
- 强调参数设定与假设透明性。

### 3.3 情景分析与压力测试
- 构建正常、悲观、极端情景参数。
- 评估不同情景下组合表现。
- 说明压力测试在风险管理与监管语境中的应用价值。

### 3.4 案例：资产收益模拟分析
- 输入历史参数与组合权重。
- 运行多路径模拟并输出分位数结果。
- 给出风险提示与资产配置建议（教学场景下）。

## 4. 实验与实践建议

- 实验 1：计算并可视化历史波动率。
- 实验 2：实现最小可用 Monte Carlo 模拟器。
- 实验 3：设计三种压力情景并比较结果差异。
