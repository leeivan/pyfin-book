# 第2章 Python 基础语法与金融实例（章节文档）

## 1. 章节定位

本章建立语法基础，把金融公式、数据输入和结果展示转化为规范的 Python 程序。

## 2. 目录结构

1. 数据类型与变量（以金融数据为例）
2. 运算符与表达式（利息、收益率计算）
3. 输入输出与格式化（金融报表展示）
4. 程序结构初步：顺序结构
5. 本章小结与练习

## 3. 内容介绍

### 3.1 数据类型与变量
- 使用金额、利率、期限等金融数据说明 `int`、`float`、`str` 的使用场景。
- 强调浮点数精度问题，并引入“结果解释时保留小数位”的习惯。
- 建立变量命名规范，如 `principal`、`rate`、`periods`。

### 3.2 运算符与表达式
- 实现单利、复利、收益率等基础计算表达式。
- 对比“数学公式写法”与“Python 表达式写法”的差异。
- 讨论括号优先级和错误表达式导致的业务偏差。

### 3.3 输入输出与格式化
- 使用 `input()` 获取参数，完成基础交互式程序。
- 使用 f-string 进行金额、百分比、对齐输出，形成报表可读性。
- 介绍最小可用的结果校验思路（输入范围、类型转换异常）。

### 3.4 顺序结构
- 明确顺序结构的执行流程。
- 用“从读取数据到输出报表”的单路径任务建立程序流程感。
- 为后续分支和循环结构做认知铺垫。

## 4. 实验与实践建议

- 实验 1：实现复利终值计算器并支持用户输入。
- 实验 2：输出“迷你资产收益报表”，包含本金、终值、收益率三列。
- 实验 3：制造一个故意错误表达式，比较错误结果与修正结果。
