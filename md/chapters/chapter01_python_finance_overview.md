## 4. 第一个金融小程序：简单收益计算

本节完成本章的第一次闭环实践，把金融公式转化为可运行代码并解释业务含义。

### 4.1 问题定义与输入输出
- 明确输入项：初始本金、期末价值；输出项：绝对收益、收益率。
- 给出计算关系：绝对收益 = 期末价值 - 初始本金；收益率 = 绝对收益 / 初始本金。
- 约定结果展示格式与边界条件说明（如初始本金为 0 的处理策略）。

### 4.2 从手算公式到代码实现
- 按“手算步骤 -> 变量设计 -> 表达式实现”的顺序完成编码。
- 解释变量命名、数据类型和表达式拆分的原因。
- 强调先保证正确性，再进行结构优化和代码美化。

### 4.3 输入校验与结果解释
- 增加输入校验，处理空值、非数字、除零等常见错误。
- 用友好提示替代程序崩溃，提升初学者调试体验。
- 解释结果的业务语义，区分“计算正确”和“业务解释正确”。
