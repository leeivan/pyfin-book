# 第1章 Python 与金融计算概述（章节文档）

## 1. 章节定位

本章是全书入口，目标是建立学习动机与工具准备能力，回答“为什么学、学什么、如何开始”。

## 2. 目录结构

1. Python 语言特点与金融应用优势
2. 金融科技（FinTech）与数据驱动金融
3. Python 开发环境搭建（Anaconda / Jupyter / IDE）
4. 第一个金融小程序：简单收益计算
5. 本章小结与学习检查

## 3. 内容介绍

### 3.1 Python 语言特点与金融应用优势
- 对比 Excel 与传统统计软件，说明 Python 在自动化、可扩展、可复用方面的优势。
- 解释“脚本化分析”和“工程化实现”的差异，建立程序化金融思维。
- 通过小例子展示“同一计算任务可以被封装并重复运行”。

### 3.2 金融科技（FinTech）与数据驱动金融
- 介绍金融科技在投研、风控、投顾和运营中的典型应用。
- 说明“数据采集 -> 数据处理 -> 模型分析 -> 业务决策”的基本链条。
- 强调课程的定位是“金融问题 + 程序实现”，而非只讲语言语法。

### 3.3 Python 开发环境搭建
- 完成 Anaconda 安装与环境管理的基本操作。
- 说明 Jupyter Notebook 与 IDE 的使用场景差异。
- 给出统一的代码命名、文件组织和运行规范，减少后续实验摩擦。

### 3.4 第一个金融小程序：简单收益计算
- 输入：初始本金、期末价值。
- 输出：绝对收益与收益率，并进行格式化展示。
- 通过“先手算后编码”方式帮助读者建立公式到代码的映射。

## 4. 实验与实践建议

- 实验 3：把脚本改造成可重复调用的函数，初步体验复用思想。
