# 第8章 常用金融数据分析库（章节文档）

## 1. 章节定位

本章引入金融数据分析主力工具链，为策略与风险章节提供计算与可视化能力基础。

## 2. 目录结构

1. NumPy：金融数值计算
2. Pandas：金融时间序列分析
3. Matplotlib：金融数据可视化
4. 案例：股票价格走势分析
5. 本章小结与分析规范

## 3. 内容介绍

### 3.1 NumPy
- 讲解数组计算、向量化操作、统计函数。
- 对比纯 Python 循环与 NumPy 计算效率。
- 通过收益率序列说明向量化在金融计算中的价值。

### 3.2 Pandas
- 介绍 `Series`、`DataFrame` 与时间索引。
- 实现按日期筛选、重采样、滚动统计等常用操作。
- 强调时间序列对齐与缺失值处理。

### 3.3 Matplotlib
- 绘制价格走势图、收益率对比图和简单指标图。
- 讲解坐标轴、图例、标题、注释等可读性元素。
- 说明可视化不是装饰，而是分析表达与沟通工具。

### 3.4 案例：股票价格走势分析
- 导入股票历史价格数据。
- 计算收益率并绘制价格与均线。
- 输出可复现的分析结论与图表文件。

## 4. 实验与实践建议

- 实验 1：用 NumPy 完成收益率批量计算。
- 实验 2：用 Pandas 完成时间序列清洗与特征统计。
- 实验 3：用 Matplotlib 输出标准化图表并附简短分析结论。
